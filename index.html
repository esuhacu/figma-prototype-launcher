Open Prototype

Tip: For the most “fullscreen” feel, add it to your Home Screen.

iPhone / iPad (best fullscreen feel)

1) Open the prototype in Safari
2) Tap Share → Add to Home Screen
3) Launch from the Home Screen icon (URL bar disappears)

Android (Chrome)

1) Tap ⋮ → Add to Home screen / Install
2) Note: Figma is not a PWA, so Chrome may still show the URL bar.
3) For true fullscreen on Android, use a WebView wrapper app.

Need true fullscreen?

Use a WebView wrapper app (no browser UI). This requires installing the app once.
