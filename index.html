Migration Onboarding Prototype PWA shell

This page is installable as a PWA shell. It can open the Figma prototype, but Figma runs on a different domain, so Android may still show browser UI after you leave this app shell.

iOS: use Share → Add to Home Screen (Apple does not allow an automatic install prompt).

Optional: Try embedded mode (may be blocked by Figma security headers).

If embedding is blocked, use the Open Prototype button instead.

Why this may not remove the URL bar on Android

PWAs only stay "standalone" inside their own scope (this GitHub Pages domain). Your prototype is on figma.com, so opening it navigates outside the PWA scope. For true fullscreen on Android, you need a native WebView wrapper app.
