Migration Onboarding Prototype Launcher

This page can be installed as an app-like launcher. On iOS, use Share → Add to Home Screen. On Android, install as a PWA.

Note: The prototype is hosted on figma.com. Android may show browser UI after navigating outside this site. For true fullscreen on Android, use the native WebView wrapper.

Tip for iPhone: You must open this launcher in Safari to add it to the Home Screen.

If you opened this from an in-app browser, tap Copy Link in the iPhone install sheet and paste into Safari.

Install on iPhone
iOS install is manual (Apple requirement)
  1. Open this page in Safari (not Chrome or an in-app browser).
  2. Tap the Share icon (square with arrow).
  3. Select Add to Home Screen.
  4. Launch from the new Home Screen icon for fullscreen mode.
